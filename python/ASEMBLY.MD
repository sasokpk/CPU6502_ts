### 🔹 Загрузка регистров

* **LDA nn** — A ← nn

* **LDX nn** — X ← nn

* **LDY nn** — Y ← nn

### 🔹 Арифметика

* **ADC nn** — A ← A + nn + C

* **SBC nn** — A ← A − nn − (1 − C)

* **MUL nn** — A ← A × nn

### 🔹 Сравнение

* **CMP nn** — установить флаги по результату A − nn

* **CPX nn** — установить флаги по результату X − nn

* **CMPC aa** — установить флаги по результату A − M[aa]

### 🔹 Логические операции

* **AND nn** — A ← A AND nn

* **ORA nn** — A ← A OR nn

* **EOR nn** — A ← A XOR nn

### 🔹 Передача данных между регистрами

* **TAX** — X ← A

### 🔹 Работа с памятью

* **STA aa** — M[aa] ← A

* **LSA aa** — A ← M[aa]

* **STX aa** — M[aa] ← X

* **LSX aa** — X ← M[aa]

### 🔹 Ввод-вывод

* **CTA** — A ← ввод с консоли

* **OTT aa** — вывод M[aa]

### 🔹 Управление флагами

* **CLC** — C ← 0

### 🔹 Условные переходы

* **BEQ label** — переход если Z = 1

* **BNE label** — переход если Z = 0

* **BCS label** — переход если C = 1

* **BCC label** — переход если C = 0

* **BMI label** — переход если N = 1

* **BPL label** — переход если N = 0

* **BVS label** — переход если V = 1

* **BVC label** — переход если V = 0

### 🔹 Безусловный переход

* **JMP label / aa** — PC ← label / aa

### 🔹 Служебные инструкции

* **NOP** — без операции

* **BRK** — останов программы

### 🔹 Метки

* **label:** — обозначение адреса программы
