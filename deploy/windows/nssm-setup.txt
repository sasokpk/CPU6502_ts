Use NSSM to run backend as Windows service.

1) Download NSSM and open elevated PowerShell.

2) Install service:
   nssm install CPU6502-WS "C:\Windows\System32\WindowsPowerShell\v1.0\powershell.exe" "-ExecutionPolicy Bypass -File C:\cpu6502\CPU6502_ts\deploy\windows\start_ws_server.ps1"

3) Set startup mode:
   nssm set CPU6502-WS Start SERVICE_AUTO_START

4) Start service:
   nssm start CPU6502-WS

5) Logs (optional):
   nssm set CPU6502-WS AppStdout C:\cpu6502\CPU6502_ts\logs\ws-out.log
   nssm set CPU6502-WS AppStderr C:\cpu6502\CPU6502_ts\logs\ws-err.log
