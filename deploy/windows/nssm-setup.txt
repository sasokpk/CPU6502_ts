Use NSSM to run Django backend as Windows service.

1) Download NSSM and open elevated PowerShell.

2) Install service:
   nssm install CPU6502-Backend "C:\Windows\System32\WindowsPowerShell\v1.0\powershell.exe" "-ExecutionPolicy Bypass -File C:\cpu6502\CPU6502_ts\deploy\windows\start_backend.ps1"

3) Set startup mode:
   nssm set CPU6502-Backend Start SERVICE_AUTO_START

4) Start service:
   nssm start CPU6502-Backend

5) Logs (optional):
   nssm set CPU6502-Backend AppStdout C:\cpu6502\CPU6502_ts\logs\backend-out.log
   nssm set CPU6502-Backend AppStderr C:\cpu6502\CPU6502_ts\logs\backend-err.log
